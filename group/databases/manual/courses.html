This document specifies how to use the courses_db.php library to interact with the student data.

1. Courses

1.1. Constructors
To create a student object, there are two static factory methods
Name Parameters Description
searchCourse
  • department_id : int
  • course_num : int
 This method searches the database for an course with the given department_id and course number.
1.1.1. Example, $course = Course::searchCourse(1,"370");

1.2. Methods
Modifier and Type Name Parameters Description
int getId Returns the unique numerical id for a course in the system. This id is only used locally.
string getdepartmentID Returns the unique numerical id for a course's department in the system. This id is only used locally.
string getFirstName
string getLastName
string getBannerId A string representing the student's 9-digit banner ID.
string getGradMonth A string in 'MM/YYYY' format.
array getMajors An array of strings. All of the values will match those in listMajors().
array getMinors An array of strings. All of the values will match those in listMinors().
void storeInDB Stores the current student in the database as a new item.
