This document specifies how to use the courses_db.php library to interact with the student data.

1. Department

1.1. Constructors
To create a department object, there are three static factory methods
Name Parameters Description
buildDepartment
  • department : string
 This is a method for creating a new department with given department name .
getDepartment
  • department : string
 This method searches the database for an existing department with the given department name.
getDepartmentById
  • id : int
 This method searches the database for an existing department with the given department id.

1.2. Methods
Modifier and Type Name Parameters Description
int getId Returns the unique numerical id for a department in the system. This id is only used locally.
string getDept Returns the department name of the current Department object.
void setDept department : string Must be a department name eg:"CS" or "MATH"
array listDepartments Ruturns array of strings representing all of the departments
void storeInDB If this is a newly created department object (made using buildDepartment()) and not yet stored in the DB, this function will create a new entry in the DB. Otherwise, this function will update the exising entry.

2. Course

2.1. Constructors
To create a course object, there are three static factory methods
Name Parameters Description
buildCourse
  • department : Department
  • course_num : int
  • title : string
 This method allows the user to specify every attribute of a new course. The values of these fields must use the same format as the corresponding entry in the courses table.
getCourse
  • department : Department
  • course_num : int
 This method searches the database for an existing course with the given Department and course number and returns the course.
getCourseById
  • id : int
 This method searches the database for an existing course with the given course id.

2.2. Methods
Modifier and Type Name Parameters Description
int getId Returns the unique numerical id for a course in the system. This id is only used locally.
Department getDepartment
int getCourseNum Returns the course number of a course object
string getTitle Returns the title of a course object. E.g., "Software Engineering"
void setDepartment department : Department
void setCourseNum course_num : int E.g., the number 370 in "CS 370"
void setTitle title : string
void storeInDB Stores current object in the databse. If the object newly created, a new entry into the DB is made. If the student has been stored in the DB, we update the existing entry

3. Semester

3.1. Constructors
To create a semeter object, there are four static factory methods
Name Parameters Description
buildSemester
  • semester : string
  • description : string
 This is a method for creating a new Semster with given semester and description.
getSemester
  • description : string
 This method searches the database for an existing semester with the given description.
getSemesterByCode
  • semester : string
 This method searches the database for an existing semester with the given semester code.
getSemesterById
  • id : int
 This method searches the database for an existing semester with the given unique id.

3.2. Methods
Modifier and Type Name Parameters Description
int getId Returns the unique numerical id for a section in the system. This id is only used locally.
string getSemesterCode
string getDescription Returns the human readable description of the semester. E.g., "Fall 2020"
void setSemesterCode string : semester Gets the internal code for the semester. E.g., "202060"
voic setDescription string : description
void storeInDB If this is a newly created Semster object (made using buildSemester()) and not yet stored in the DB, this function will create a new entry in the DB. Otherwise, this function will update the exising entry.

4. Section

4.1. Constructors
To create a Section object, there are four static factory methods
Name Parameters Description
buildSection
  • course : Course
  • semester : Semester
  • section : int
  • crn : string
 This is a method for creating a new Section with given Semester, Course, section and crn.
getSection
  • course : Course
  • semester : Semester
  • section : int
 This method searches the database for an existing section with the given Course, semester and section number.
getSectionByCrn
  • crn : string
  • semeter : Semester
 This method searches the database for an existing section with the given crn and Semester
getSectionById
  • id : int
 This method searches the database for an existing section with the given unique id.

4.2. Methods
Modifier and Type Name Parameters Description
int getId Returns the unique numerical id for a section in the system. This id is only used locally.
Course getCourse Returns Course object of the current section.
Semester getSemester Returns semester object of cuurent section
int getSectionNum
string getCrn Returns a string representing the crn of the section object
void setCourse course : Course
void setSemster semester : Semester
void setCrn crn : string Must be string with four characters
void setSectionNum section : int
void storeInDB If this is a newly created Section object (made using buildSection()) and not yet stored in the DB, this function will create a new entry in the DB. Otherwise, this function will update the exising entry.
Example
Here, we demonstate some use cases of all the classes. 

// First, we create a new department object
$cs_dept = Department::buildDepartment("CS");

//We store the department in the database
$cs_dept->storeInDB();

//Now, we create new course object
$dataStructures = Course::buildCourse($math_dept, 310, "Data Structures and Algorithms");

//We store the course object in the table
$dataStructures->storeInDB();

//We get a semester by using the semester code
$fallsem = Semester::getSemesterByCode("202060");

//finally we can create a section
$csSection = Section::buildSection($dataStructures, $fallsem, 1, "1234");

//We can retrieve a section using crn
$section = Section::getSectionByCrn($fallsem, "1234");

/We can also retrieve a section using id.
$section1 = Section::getSectionById(1);
